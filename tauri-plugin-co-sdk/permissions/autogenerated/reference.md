## Default Permission

Allows usage of co-sdk api

#### This default permission set includes the following:

- `allow-get-co-state`
- `allow-resolve-cid`
- `allow-storage-get`
- `allow-storage-set`
- `allow-push-action`
- `allow-get-actions`
- `allow-create-identity`
- `allow-session-open`
- `allow-session-close`
