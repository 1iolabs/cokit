# Tauri Integration

## Installation

- Install NodeJS
- `cd tauri-plugin-co-sdk`
- `npm install`
- `../json-schemas/generate-ts.sh`
- `cd examples/messenger`
- `npm install`

## Build Messenger Demo Application

- `cd tauri-plugin-co-sdk/examples/messenger`
- `npm run build-dist`
- `npm run tauri-build`

### Run in Developer Mode

- `npm run tauri-dev`
