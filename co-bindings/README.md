# co-bindings

## Abstract

This crate contains wrappers for variaous co-sdk types for that uniffi and flutter_rust_bridge bindings can be generated.

## Features
- [x] Flutter: Initial bridge
- [x] Flutter: DagCbor encoding
- [x] Flutter: Initial flutter high level components
- [x] Initial CoMap
- [ ] CoMap: Transaction
- [ ] Initial CoSet
- [ ] Initial CoList
- [ ] More primitives
- [ ] Integrate `co-js`

## Flutter 

### Setup

```sh
cargo install flutter_rust_bridge_codegen@2.11.1
```

### Build

```sh
./co-bindings/flutter/build.sh
```
