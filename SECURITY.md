# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in COKIT, please report it
privately. **Do not open a public issue.**

Email: **security@1io.com**

We will:
- Acknowledge receipt within 48 hours
- Provide an initial assessment within 5 business days
- Work with you on responsible disclosure

## Scope

This policy covers the `cokit` repository. For GUARD-related security
concerns, contact the same address.

## Supported Versions

Only the latest release is actively maintained with security fixes.
